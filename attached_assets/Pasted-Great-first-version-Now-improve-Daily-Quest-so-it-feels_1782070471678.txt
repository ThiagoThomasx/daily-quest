## Main goals

1. Improve the Today screen so it feels more engaging.
2. Add a strong post-completion state.
3. Improve the Skip/Reroll experience.
4. Make the app more PWA/mobile friendly.
5. Add better visual hierarchy and microinteractions.

## Today screen improvements

The Today screen should have two different states:

### State 1 — Active quest

Show:

- Date
- App title
- Current streak
- Level and XP progress
- Today's quest card
- Category badge
- Difficulty badge
- Estimated time
- XP reward
- Main CTA: Complete Quest
- Secondary CTA: Skip / Reroll

Improve the quest card so it feels more premium:

- More visual depth
- Better spacing
- Stronger typography
- Better category/difficulty badges
- Clearer XP reward
- More satisfying CTA area

### State 2 — Quest completed

After the user completes today’s quest, replace the active challenge card with a completed state.

Show:

- A celebration message
- XP gained
- Updated streak
- Updated level progress
- A “Come back tomorrow” message
- A small summary of today’s completed quest
- Optional button: View History

Suggested copy:

“Quest Complete”
“You earned +25 XP”
“Streak updated. Come back tomorrow for your next quest.”

Do not allow the same quest to be completed twice.
